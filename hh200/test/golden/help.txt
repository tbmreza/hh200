Run hh200 scripts

Usage: hh200 [COMMAND | [SOURCE] [-V|--version] [-F|--debug-config] [-C|--call] 
               [-R|--rps] [-S|--shotgun N] [-d|--lsp PORT] [--lsp-stdio]]

Available options:
  SOURCE                   Path of source program
  -V,--version             Print version info and exit
  -F,--debug-config        Read environment and script header to determine the
                           config values without executing script's side-effects
  -C,--call                Execute a script snippet directly
  -R,--rps                 Start "requests per second" mode
  -S,--shotgun N           Execute in N parallel workers at once (default: 1)
  -d,--lsp PORT            Run hh200 language server
  --lsp-stdio              Run hh200 language server over stdio
  -h,--help                Show this help text

Available commands:
  browse                   Launch the dashboard
