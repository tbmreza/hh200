# Neovim/Vim Syntax Highlighting for hh200

This directory contains the Vim syntax highlighting files for `hh200`.

## Installation

### Manual Installation

Copy the files to your `~/.config/nvim/` (or `~/.vim/`) directory:

```bash
cp -r editors/nvim/* ~/.config/nvim/
```

### Plugin Managers

You can also point your plugin manager to this directory if you structure it as a plugin.

## Usage

Open a file with `.hh200` extension, and syntax highlighting should be automatically applied.
